# Endless Runner

A 2D endless runner game built with Unity.

## Description

Endless Runner is a simple game where the player controls a character that runs automatically, jumping over obstacles to achieve a high score. Collect coins along the way!

## Features

- **Main Menu**: Start the game or quit.
- **Gameplay**: Endless running with jumping mechanics, score tracking, and coin collection.
- **Game Over Panel**: Displays final score and coins, with option to restart.

## Requirements

- Unity 2021.3.45f2 or later

## How to Run

1. Open the project in Unity Editor.
2. Open the MainMenu scene from Assets/Scenes/.
3. Press Play.

## Controls

- **Jump**: Spacebar or Left Mouse Click

## Gameplay

- The player runs automatically to the right.
- Jump to avoid obstacles.
- Collect coins for points.
- Game ends when health reaches zero or falling off screen.
